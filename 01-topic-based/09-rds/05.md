# Imagine You Own a Library 📚

You have **one librarian**.

People come all day.

There are only **2 kinds of work**.

## Person 1

> I want to borrow a book.

This changes the records.

This is a **WRITE**.

---

## Person 2

> Where is Harry Potter?

The librarian just looks it up.

Nothing changes.

This is a **READ**.

---

# Now Imagine 10,000 Students Arrive 😱

Every student asks:

# What should you do?

### Option A

Hire another librarian to answer questions.

```text
Main Librarian

Helper Librarian
```

Perfect.

Now both answer students.

This is exactly what **Read Replica** does.

It helps with **READS**.

---

# Another Situation

Your librarian suddenly gets sick.

Can students borrow books?

No.

So you hire a backup librarian.

```text
Main Librarian

Backup Librarian
```

The backup does nothing all day.

He waits.

If the main librarian gets sick...

He immediately starts working.

This is **Multi-AZ**.
